How to use the ladder

Quick tips

Sign up

Open a new GitHub issue with the title:

Signup: <Your_Warzone_ID> Name: <Your_Warzone_Username>

Example: Signup: 1234567 Name: General_Risk

Update your game cap

Limits your simultaneous ladder games (default 2, max 3, set to 0 to pause).

Update: <Your_Warzone_ID> Cap: <GameLimit>

Example: Update: 1234567 Cap: 3

Remove yourself

Remove: <Your_Warzone_ID>

Example: Remove: 1234567

Add a map template (Admin)

Adds a Warzone template to the ladder's random map pool.

AddTemplate: <Template_ID> Name: <Map_Name>

Example: AddTemplate: 1390041 Name: Strategic 1v1

Remove a map template (Admin)

RemoveTemplate: <Template_ID>

Example: RemoveTemplate: 1390041

What to expect

  • The bot checks the ladder every few hours to create or clean up games.
  • Missing two games in a row will mark you inactive until you join another match.
  • You can always re-activate by joining a game or updating your status.
