How to join and stay active

GitHub Issues

Join the ladder

Open a new GitHub issue with the title:

Signup: <Your_Warzone_ID> Name: <Your_Warzone_Username>

Example: Signup: 1234567 Name: General_Risk

Update your game cap

Change how many active games you want (0 pauses you, 3 is max):

Update: <Your_Warzone_ID> Cap: <GameLimit>

Example: Update: 1234567 Cap: 3

Remove yourself

If you're done playing, request removal with:

Remove: <Your_Warzone_ID>

Example: Remove: 1234567

What the bot does

Matchmaking & cleanup

Automated
  • Matchmake every 2 hours: finds two available players and creates a Warzone game.
  • Referee completed games: records results, updates ELO, and tracks missed games.
  • Strike rules: two missed games mark you inactive; playing again resets strikes.
  • Lobby cleanup: games stuck in lobby > 3 days are deleted; no-join players get a strike.
